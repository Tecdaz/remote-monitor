## Key takeaways

- **Current target = Wear OS 7** (Android 17, API 37). The `versions/latest` alias always points here.
- **Headline Wear OS 7 capabilities**: dynamic Wear Widgets (replacing full-screen Tiles) and Live Updates.
- **Quality bar**: every Wear app must pass the checklist in `app-quality.md` before Play submission.
