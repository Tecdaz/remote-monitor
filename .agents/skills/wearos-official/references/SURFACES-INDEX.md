## How to use this catalog

1. Pick the surface (Tile / Complication / Widget / Notification / Watch Face / App).
2. Read the 1-line description to confirm the page is the right one.
3. Open the canonical URL for full content.
4. If a per-page summary exists, read it first — it captures the page's structure in 4 sections (purpose / sections / when to consult / source quote).

## Surface decision recap

When in doubt, the surfaces taxonomy (`surfaces.md`) gives the heuristic: pick the surface whose main job matches your experience's purpose.

| Surface | Use when |
| --- | --- |
| **App** | Interactive, full-screen session |
| **Tile** | Quick, glanceable, non-scrollable (deprecated path) |
| **Widget** | Quick, glanceable card (Wear OS 7+ replacement for tiles) |
| **Complication** | Persistent at-a-glance data on the watch face |
| **Watch face** | Always-on top-level experience |
| **Notification** | High-value, contextual, asynchronous |
