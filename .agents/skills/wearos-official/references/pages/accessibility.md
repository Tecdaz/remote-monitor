# Accessibility on Wear OS

**URL**: https://developer.android.com/training/wearables/accessibility
**Section**: Get started → Quality & a11y

## What this page is

The official Wear-specific **accessibility guide** — layered on top of the Android accessibility guide. Focuses on Wear-specific considerations: small screen, rotary input, font scaling.

## Sections

- Support the user's preferred font size
- Enable rotary input
- Optimize your app for Talkback
- Set minimum touch targets

## When to consult

Before shipping any Wear UI. Companion to `design-type-accessibility.md` (typography-specific accessibility).

## Source quote

> Wear OS apps have some additional considerations when it comes to accessibility due to the following factors: small screen, limited input methods, ambient contexts.
