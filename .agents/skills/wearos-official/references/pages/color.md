# Color (Wear OS)

**URL**: https://developer.android.com/design/ui/wear/guides/styles/color
**Section**: Design → Styles

## What this page is

The Wear OS color guide — principles for building themes: starting from black, the new color roles, semantic meaning, and accessibility (contrast).

## Sections

- Build from black (dark-first, OLED-friendly)
- New color roles (Material 3 Expressive tokens)
- Semantic meaning
- Color accessibility (contrast compliance)
- What's new
- Resources

## When to consult

When defining a theme, choosing color tokens, or auditing contrast on a Wear UI. Especially relevant for OLED displays where black backgrounds save power.

## Source quote

> Create more accessible, personal color schemes communicating your product's hierarchy, state, and brand. When designing for wearables, color plays a crucial role in enhancing legibility, usability, visual appeal, and expression, especially on smaller screens.
