# Common adaptive design layouts

**URL**: https://developer.android.com/design/ui/wear/guides/foundations/common-layouts
**Section**: Design → Layouts

## What this page is

Catalog of the **canonical Wear OS layouts** — proven, versatile patterns that scale across the variety of watch shapes (small/large round, square). Use these as starting points instead of inventing layouts.

## Sections

- Use established common layouts

## When to consult

When picking the layout for a new screen (list, detail, button screen, etc.). Reach for the canonical pattern first; only diverge with a strong reason.

## Source quote

> Common design layouts—also known as canonical design layouts—are proven, versatile app layouts that provide an optimal user experience on the wide variety of screen sizes that are available with Wear OS.
