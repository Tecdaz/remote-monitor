# Compose for Wear OS Codelab

**URL**: https://developer.android.com/codelabs/compose-for-wear-os
**Section**: Get started → Hands-on

## What this page is

The official **hands-on codelab** for Compose on Wear OS — walks through building a complete Wear app from the "Empty Wear App" template: dependencies, MainActivity, simple composables, lists, theming. Best way to internalize the API by doing.

## Sections

- About this codelab
- 1. Introduction (what you'll learn / build / prerequisites)
- 2. Getting Set Up (download code, open in Android Studio, explore start)
- 3. Review the dependencies
- 4. Review MainActivity
- 5. Add Simple Composables (Icon Button, Text)
- ... (continues with lists, theming, navigation)

## When to consult

When onboarding to Compose for Wear OS — the most efficient path from "never built a Wear app" to "shipped a working screen". Better than reading the API reference cold.
