# Jetpack Compose performance on Wear OS

**URL**: https://developer.android.com/training/wearables/compose/performance
**Section**: Get started → Compose → Performance

## What this page is

The official Wear-specific Compose **performance guide** — composes the Compose performance techniques that matter most on resource-constrained watches. Especially important for Material 3 Expressive's richer animations.

## Sections

- Essential performance improvement techniques
- Performance Measurement and Validation (release-mode measurements, Macrobenchmark)
- Additional tools
- Other resources

## When to consult

Before shipping, or whenever animations feel sluggish. Companion to `power.md` — performance on Wear is battery on Wear.

## Source quote

> Performance on Wear OS is an essential consideration for apps, as many Wear OS devices have limited CPU and GPU resources compared to larger mobile devices. With the introduction of richer animations and dynamic effects in Material 3 Expressive, you should validate and improve the performance of your app's key workflows.
