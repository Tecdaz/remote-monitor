# Use Jetpack Compose on Wear OS

**URL**: https://developer.android.com/training/wearables/compose
**Section**: Get started → UI framework

## What this page is

The official guide to Compose for Wear OS — explains what's similar to mobile Compose, what differs, and why this is the recommended UI framework for Wear apps. Covers surfaces (App, Tile, Watch Face), Material Design integration, and compatibility.

## Sections

- Material Design in Jetpack Compose on Wear OS
- Compatibility (with mobile Compose, version mapping)
- Surfaces (App vs Tile vs Watch Face, one Compose runtime per surface)
- Setting up (dependencies, versions)
- Feedback channels

## When to consult

When picking the UI framework for a new Wear app, when migrating from Views, or when targeting a Wear-specific surface. Also useful to understand the version mapping when reading dependency errors.

## Source quote

> Compose for Wear OS is part of Android Jetpack... it helps you write better code faster. This is our recommended approach for building user interfaces for Wear OS apps.
