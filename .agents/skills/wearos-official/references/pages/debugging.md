# Debug a Wear OS app

**URL**: https://developer.android.com/training/wearables/get-started/debugging
**Section**: Get started → Test & debug

## What this page is

The official **debugging** entry point for Wear OS apps — same standard tools as other Android form factors, with Wear-specific setups (USB / Wi-Fi).

## Sections

- Debug targets (emulator vs physical watch)
- Additional debugging and setup (links to wifi debug, screenshots, etc.)

## When to consult

First time debugging on a watch, or when standard mobile debugging doesn't apply.

## Source quote

> Debugging on Wear OS uses the same standard tools and processes as debugging on other Android-powered form factors.
