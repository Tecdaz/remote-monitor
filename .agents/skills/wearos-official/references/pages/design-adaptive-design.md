# Adaptive design — Overview

**URL**: https://developer.android.com/design/ui/wear/guides/foundations/adaptive-design
**Section**: Design → Foundations → Adaptive

## What this page is

The overview page for **adaptive & responsive design** on Wear OS. Defines essential terms, layout types, key screen sizes, and quality tiers. Critical reading for any UI that must work across the variety of Wear form factors (small round, large round, square).

## Sections

- Essential terms (responsive vs adaptive design)
- Types of layouts (responsive layout, adaptive layout)
- Key screen sizes (the canonical set of watch sizes)
- Design quality tiers (good → better → best)

## When to consult

First stop when designing for multiple watch sizes. Read this before touching `quality-tiers` sub-pages.

## Source quote

> The Wear OS ecosystem consists of devices with a wide variety of screen sizes. Using adaptive UI principles is critical to delivering the highest quality experience for everyone. Adaptive UIs stretch and change to make the best possible use of all available screen space.
