# Best practices for designing apps

**URL**: https://developer.android.com/design/ui/wear/guides/surfaces/apps/best-practices
**Section**: Design → Surfaces → Apps → Best practices

## What this page is

Concrete, opinionated best-practices list for the **App surface**. The eight rules every Wear app should follow: vertical layouts, show the time, inline entry points, labels, elevate primary actions, scrollbars on scrolling screens, responsive design, percentage margins.

## Sections

- Optimize for vertical layouts
- Show the time (top overlay)
- Accessible inline entry points
- Use labels to orient users
- Elevate primary actions
- Show the scrollbar on scrolling screens
- Design responsively for larger screen sizes
- Use responsive (percentage) margins

## When to consult

Review checklist for any app screen. These eight rules are the "did you remember the obvious?" checklist.

## Source quote

> Simplify your app's design by using vertical layouts, which allow users to scroll in a single direction to traverse content.
