# Figma Design Kits

**URL**: https://developer.android.com/design/ui/wear/guides/get-started/design-kits
**Section**: Design → Get started → Kits

## What this page is

The official **Figma kits** for Wear OS design — pre-built components, styles, variables, and layouts you can drop into a Figma file to start designing apps, tiles, and media screens without rebuilding everything from scratch.

## Sections

- Design kit for apps (Figma file with components, styles, variables, layouts)
- Design kit for tiles (Figma file for tile layouts)

## When to consult

Before starting any new design work in Figma. Open the relevant kit first — saves days of rebuilding Material 3 Expressive components.

## Source quote

> Use Figma design kits to create your own wearable apps, tiles, media screens, and flows. These kits include components, styles, variables, and layouts to kickstart your designs.
