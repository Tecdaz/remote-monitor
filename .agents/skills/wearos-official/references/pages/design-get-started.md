# Material 3 Expressive — Design system overview

**URL**: https://developer.android.com/design/ui/wear/guides/get-started

## What this page is

The official landing page for **Material 3 Expressive** on Wear OS — the current design system, optimized for the round watch form factor. Explains the high-level principle: elevate each UI layer with thoughtful shape, animation, color, and typography.
