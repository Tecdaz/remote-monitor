# Common layouts — Scrolling apps

**URL**: https://developer.android.com/design/ui/wear/guides/foundations/common-layouts/apps-scrolling
**Section**: Design → Common layouts → Scrolling

## What this page is

The canonical layouts for **scrolling app screens** — lists (`TransformingLazyColumn`) and dialogs. The majority of app screens fit this pattern. Explains the responsive/adaptive behavior the Wear Compose components ship with.

## Sections

- Build responsive and optimized designs
- Build adaptive and differentiated designs
- Responsive and adaptive behavior
- Create differentiated experiences

## When to consult

When designing any screen that scrolls — list screens, dialog flows, multi-step tasks. Use this to pick the canonical pattern and to understand the built-in responsive behavior of Wear Compose components.

## Source quote

> Scrolling app view layouts include lists (TransformingLazyColumn) and dialogs. These layouts make up the majority of app screens, and they represent a collection of components which need to adapt to larger screen sizes.
