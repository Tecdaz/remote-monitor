# Media design principles

**URL**: https://developer.android.com/design/ui/wear/guides/patterns/media/principles
**Section**: Design → Patterns → Media

## What this page is

The **media-app-specific** design principles for Wear OS — four principles plus the guidance to reuse patterns rather than invent new ones.

## Sections

- Consistent and predictable
- Glanceable and critical
- Common design patterns
- Extend across devices

## When to consult

When designing any media app (audio, video, podcast) on Wear. Read alongside the general Wear principles (`principles.md` + `design-m2-5-principles.md`).

## Source quote

> Design media experiences on Wear OS by applying core principles that prioritize user control and efficiency. Align UI patterns and avoid creating new UIs for common interactions.
