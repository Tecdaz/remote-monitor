# Tier 3: Adaptive and differentiated

**URL**: https://developer.android.com/design/ui/wear/guides/foundations/quality-tiers/adaptive-differentiated
**Section**: Design → Quality tiers → Tier 3

## What this page is

Defines the **highest quality tier** — adaptive UIs that use breakpoints to deliver size-specific experiences (e.g., layouts that small screens can't match). Critical detail: the canonical breakpoint is **225dp** for Wear OS.

## Sections

- Use breakpoints on Wear OS (225dp breakpoint recommendation)
- Examples

## When to consult

When designing for the largest watch sizes or when you want to ship a "best-in-class" experience. The 225dp number is the key takeaway — use it as your breakpoint.

## Source quote

> Apps that create a user experience specifically tailored to the size of the screen, leveraging new layouts, implemented at a key screen size breakpoint, to derive additional value for users of devices with larger screens.
