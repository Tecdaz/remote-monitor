# Tier 1: Ready for all screens

**URL**: https://developer.android.com/design/ui/wear/guides/foundations/quality-tiers/ready-all-screens
**Section**: Design → Quality tiers → Tier 1

## What this page is

Defines the **minimum quality tier** for Wear OS UIs — your app must work on all supported screen sizes without obvious bugs or interaction issues, even if the layout isn't optimal.

## Sections

- Create a usable UI (no clipping, no broken flows)
- Examples

## When to consult

As the first quality gate. If your app isn't Tier 1 (working everywhere), it isn't ready to ship — no matter how polished Tier 2/3 work is on the main target device.

## Source quote

> This tier represents a minimum quality bar for experiences across all screen sizes on Wear OS. The first step in creating a great app for Wear OS is making your app ready for all screen sizes.
