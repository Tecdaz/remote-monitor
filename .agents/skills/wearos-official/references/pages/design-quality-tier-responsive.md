# Tier 2: Responsive and optimized

**URL**: https://developer.android.com/design/ui/wear/guides/foundations/quality-tiers/responsive-optimized
**Section**: Design → Quality tiers → Tier 2

## What this page is

Defines the **second quality tier** — responsive layouts that automatically adapt to different screen sizes, offering more text, actions, or larger tap targets on larger screens. Examples included.

## Sections

- Add value through responsive design
- Examples

## When to consult

When grading your own UI's responsiveness. Tier 2 is the bar to clear for shipping a "good" Wear app.

## Source quote

> Apps that employ responsive layouts and automatically adapt to different screen sizes offer additional value to users and provide more productive, engaging user experiences.
