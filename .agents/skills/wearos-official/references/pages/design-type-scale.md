# Type scale (Wear OS)

**URL**: https://developer.android.com/design/ui/wear/guides/styles/typography/type-scale-tokens
**Section**: Design → Styles → Typography → Type scale

## What this page is

The complete **Material 3 Expressive type scale** — 21 styles in six families (Display, Title, Label, Body, Numeral, Arc). Default typeface is **Roboto Flex**. Each style has an intended application and meaning.

## Sections

- Display (large, short, highly glanceable)
- Title
- Label
- Body
- Numeral (tabular numeric displays)
- Arc (curved/circular text)

## When to consult

When picking a type style for a screen element. Reach for the right role before falling back to "title large" — the scale exists to give you the right size/weight for each context.
