# Wear OS gallery — Sample apps

**URL**: https://developer.android.com/wear/gallery
**Section**: Essentials → Samples

## What this page is

Curated showcase of **Wear OS apps** — use cases (fitness, messaging, payments, media, etc.) and UI patterns you can study to see what good looks like on the platform. Acts as a design reference for inspiration, not as source code.

## When to consult

When you need visual/UX inspiration before designing a new feature, or to validate a pattern against real shipped apps.
