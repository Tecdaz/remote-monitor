# Create and run your first Wear OS app

**URL**: https://developer.android.com/training/wearables/get-started/creating
**Section**: Get started → Create

## What this page is

The official **end-to-end setup tutorial** — the first thing to read or do. Covers creating a project from the Empty Wear App template, configuring an emulator, deploying to a physical watch, and the architecture-planning questions to lock in BEFORE adding features.

## Sections

- Create a Wear OS app (New Project → Wear OS → Empty Wear App)
- Run an app on the emulator (Device Manager → Wear OS Small Round)
- Run an app on a physical watch (enable ADB, USB/Wi-Fi)
- Plan your app architecture (app model, UI, data, long-running, surfaces, power)

## When to consult

First time you build a Wear app, or when onboarding a new dev. The architecture-planning subsection is the single most important pre-code checklist.
