# Notifications on Wear OS

**URL**: https://developer.android.com/training/wearables/notifications
**Section**: Surfaces → Notifications

## What this page is

The official guide to **notifications on Wear** — uses the same APIs as phone notifications but with wearable-specific extensions (previews, actions, inline reply, bridges). Covers how notifications appear (bridged from phone or standalone), recommended patterns, and avoiding duplicate notifications.

## Sections

- Recommended notifications for wearables
- Avoid duplicate Notifications
- Add wearable-specific features to a notification
- Launch your phone app from a wearable device

## When to consult

When building any feature that surfaces information asynchronously (messages, alerts, status updates). Notifications are a Wear surface in their own right, not just a fallback.

## Source quote

> Notifications on watches use the same APIs and have the same structure as notifications on phones.
