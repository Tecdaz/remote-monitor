# Request permissions on Wear OS

**URL**: https://developer.android.com/training/wearables/apps/permissions
**Section**: Get started → Apps

## What this page is

The official guide to **permissions on Wear** — same APIs as mobile, with extra Wear-specific scenarios (services, ongoing activities, etc.).

## Sections

- Permission scenarios
- Permission-request patterns
- Permissions for services
- Settings

## When to consult

Before requesting any runtime permission in a Wear app — especially for sensors, location, body sensors, microphone, or notifications. The UX flow differs from mobile (small screen = different prompt density).

## Source quote

> Requesting permissions on Wear OS is similar to requesting permissions in mobile apps, with a couple of additional use cases. This document assumes you understand how Android permissions work.
