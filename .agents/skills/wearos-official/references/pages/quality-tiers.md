# Design quality tiers

**URL**: https://developer.android.com/design/ui/wear/guides/foundations/quality-tiers
**Section**: Design → Responsive & adaptive

## What this page is

Defines the **quality tiers** that a Wear UI must hit on different screen sizes and shapes, plus the rules for **UI element scaling** so layouts feel native on small round, large round, and square form factors.

## Sections

- Maintain UI element scaling (size classes, round vs square)
- Tiers of quality guidelines (good / better / best)

## When to consult

When designing for multiple watch sizes/shapes. Use this to grade a UI's responsiveness before shipping.
