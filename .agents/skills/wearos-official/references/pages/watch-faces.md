# Build watch faces

**URL**: https://developer.android.com/training/wearables/watch-faces
**Section**: Surfaces → Watch faces

## What this page is

The official watch faces hub. Since **January 2026 the Watch Face Format is required** for installing watch faces on all Wear OS devices — the legacy `WatchFaceService` path is being phased out.

## Sections

- Building watch faces (entry point; defers to Watch Face Format docs and Watch Face Studio)

## When to consult

Any time you need to build a watch face. Today this means: Watch Face Studio (no-code tool) or the **Watch Face Format** (declarative XML/JSON). Do not start a new `WatchFaceService`-based project.

## Source quote

> As of January 2026, the Watch Face Format is required for installing watch faces on all Wear OS devices.
