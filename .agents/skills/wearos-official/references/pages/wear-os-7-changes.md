# Wear OS 7 — Behavior changes

**URL**: https://developer.android.com/training/wearables/versions/7/changes
**Section**: Essentials → Versions → Wear OS 7

## What this page is

The official list of **behavior changes** in Wear OS 7 (based on Android 17, API 37). Critical for testing — many changes affect all apps running on Wear OS 7, not just newly-targeted ones.

## Sections

- Background audio hardening
- Local network access permissions

## When to consult

Before targeting Wear OS 7 or testing your app on a Wear OS 7 device. Each change lists what to test.

## Source quote

> Wear OS 7 is based on Android 17 (API level 37). When you prepare your Wear OS app for use on Wear OS 7, handle the system behavior changes that affect all apps in Android 17, as well as the changes for apps that target Android 17.
