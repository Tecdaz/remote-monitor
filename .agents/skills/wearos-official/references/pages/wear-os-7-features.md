# Wear OS 7 — Features

**URL**: https://developer.android.com/training/wearables/versions/7/features
**Section**: Essentials → Versions → Wear OS 7

## What this page is

The official overview of **new features in Wear OS 7**. Headline: legacy full-screen Tiles evolve into dynamic **Wear Widgets**; plus **Live Updates** for real-time status.

## Sections

- Wear Widgets (new glanceable surface)
- Live Updates (real-time status notifications)

## When to consult

When planning a Wear OS 7-targeted feature. These are the major capabilities worth leveraging.

## Source quote

> Starting with Wear OS 7, legacy full-screen Tiles are evolving into dynamic Wear Widgets. Widgets are a new glanceable surface for Wear OS, designed to complement apps and watch faces.
